隐私政策 · Privacy Policy

Web Watermark Tool · 网页水印工具
Last updated: 2026-07-14
中文 English

1. 数据收集

网页水印工具（Web Watermark Tool）不收集、不传输、不共享任何个人数据、使用行为数据或浏览记录。所有功能均在你的浏览器本地运行。

2. 数据存储

用户配置（匹配规则、水印文字、颜色等）通过 Chrome 内置的 chrome.storage.sync 机制存储。这份数据：

  • 由 Google 在传输与存储过程中加密
  • 仅在你登录同一 Google 账号的多台 Chrome 实例之间同步
  • 本扩展作者及任何第三方均无权访问
  • 可通过 Chrome 设置随时清除

3. 权限说明

本扩展在 manifest.json 里申请的权限：

  • storage —— 用于把你配置的水印规则持久化保存
  • activeTab —— 读取当前活动 tab 的 URL，用于在工具栏图标上显示对应环境的短标签（如 PROD、TEST）

Content script 通过 <all_urls> 匹配所有 URL，仅用于按用户自定义规则叠加水印。不读取、不上传任何页面内容。

4. 第三方服务

本扩展不与任何外部服务器通信，不使用 CDN、不加载远程脚本、不接入分析服务（如 Google Analytics、Sentry 等）、不接入广告 SDK。

5. 儿童隐私

本扩展面向开发者与运维人员，不主动收集任何用户信息，因此也不针对 13 岁以下儿童收集数据。

6. 变更

本政策若有更新，将在本页面直接修改。建议定期查看本页面获取最新版本。

7. 联系方式

1. Data Collection

The Web Watermark Tool extension does NOT collect, transmit, or share any personal data, usage data, or browsing history. All operations happen locally in your browser.

2. Data Storage

User configurations (rules, watermark text, colors, opacity, etc.) are stored using Chrome's built-in chrome.storage.sync mechanism. This data is:

  • Encrypted in transit and at rest by Google
  • Synchronized only between your own logged-in Chrome instances
  • Never accessible to the extension author or any third party
  • Removable at any time via Chrome settings

3. Permissions Rationale

Permissions requested in manifest.json:

  • storage — to persist your watermark configurations across sessions
  • activeTab — to read the currently active tab's URL for updating the toolbar icon badge with the environment short label (e.g., PROD, TEST)

The content script's <all_urls> match pattern is used solely to inject the watermark visual layer on any URL the user chooses to watermark via their own rules. Page content is neither read nor transmitted.

4. Third-Party Services

This extension does NOT communicate with any external server. No CDN, no remote script loading, no analytics service (Google Analytics, Sentry, etc.), no advertising SDK.

5. Children's Privacy

This extension is designed for developers and operations engineers. It does not knowingly collect any information from anyone, including children under 13.

6. Changes

Updates to this policy will be reflected directly on this page. Please check periodically for the latest version.
